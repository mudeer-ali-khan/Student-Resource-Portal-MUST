# Student Resource Portal - BrightFuture Learning Center

This is a responsive web application developed as part of the **Web Engineering (Theory)** course at **Mirpur University of Science and Technology (MUST)**.

The "Student Resource Portal" is a central dashboard designed for students to access course materials, stay updated with campus announcements, and manage their academic profiles. 

### Key Features
* **Semantic HTML5:** Built using modern semantic tags (`<header>`, `<nav>`, `<main>`, `<aside>`, `<footer>`) for better SEO and accessibility.
* **Grid Layout:** Utilizes **CSS Grid** for the main structural layout (content area vs. sidebar).
* **Flexbox Features:** Uses **Flexbox** for the navigation bar and the responsive feature cards.
* **Responsive Design:** Fully mobile-friendly with media queries that adjust the layout for smaller screens.
* **Course Directory:** A styled HTML table showcasing available courses and instructors.
